ZegesMenden/Projects/

Thrust vector control flight computers.

Since December 2020, I have been working on avionics systems for model rockets, and have been focusing on flight computers for thrust vector control (tvc).

My flight computers started out simple - only using breakout board from amazon and adafruit. Over time they got more and more complex,

eventually using exclusively surface mount components, with all the microcontrollers and sensors soldered directly to the boards.
